## Задача 2. ПИД-регуляторы
------------------------------------------------------
Необходимо реализовать программу на языке C++, которая моделирует ПИД-регулятор. В качестве объекта управления используется математическая модель, разработанная в предыдущей задаче. Программа должна быть основана на принципах объектно-ориентированного программирования и содержать как минимум три класса с использованием наследования. Результаты работы программы должны быть представлены в виде графиков для различных температурных параметров объекта, с пояснениями (отчет составляется в формате .md с помощью Doxygen).
------------------------------------------------------
### Реализация
В программе реализовано четыре класса:
1. `TempEquation` — абстрактный класс для температурных уравнений.
2. `SimpleTempEquation` — класс для простой модели.
3. `ComplexTempEquation` — класс для сложной модели.
4. `ControlUnit` — класс, реализующий логику ПИД-регулятора.
------------------------------------------------------

Для ПИД-регулятора установлены следующие параметры: **gain = 0.1, integrationTime = 10, derivativeTime = 80, samplingInterval = 50, simulationTime = 30**.
